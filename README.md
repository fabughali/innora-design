## 🚀 Getting Started

### For Developers
```bash
# Clone the repository
git clone https://github.com/fabughali/innora-design.git
cd innora-design

# Install dependencies
flutter pub get

# Run in development mode
flutter run -d chrome

# Build for production
flutter build web --release
```

### For Users
Simply visit [https://fabughali.github.io/innora-design/](https://fabughali.github.io/innora-design/) to experience the luxury of INNORA's digital presence.
